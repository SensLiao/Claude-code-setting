# Living Canvas — 六条铁律（gate 形式）

任何 Living Canvas 相关 artifact 必须每条都打勾才放行。详见 `workflows/living-canvas-interaction.md` 完整说明。

- [ ] **铁律 1**：操作语义传统（菜单只出现 Create/Duplicate/Delete/Group/Move/Zoom 等标准词）
- [ ] **铁律 2**：创意只在反馈层，不在信息架构层（keyboard / 选择模型 / 复制语义可预测）
- [ ] **铁律 3**：动画 ≤500ms（micro 150-300 / 复杂 ≤400）
- [ ] **铁律 4**：`prefers-reduced-motion` 兜底齐全
- [ ] **铁律 5**：不可阻塞操作（cancel / interrupt 钩子齐全；不冻结 pointer event）
- [ ] **铁律 6**：不污染主视觉（glow/shimmer/bloom/particle 每屏 ≤2 处）
